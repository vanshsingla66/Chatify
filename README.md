# Chatify

Chatify is a full-stack real-time language exchange platform where users can find partners, chat, and make video calls.

## Features
- JWT Authentication
- User onboarding
- Friend requests
- Real-time chat
- Video calling
- Theme customization

## Tech Stack

Frontend:
- React
- Zustand
- React Query
- TailwindCSS
- Stream Chat SDK

Backend:
- Node.js
- Express
- MongoDB
- JWT Authentication

## Project Structure

Chatify/
├── backend/
├── frontend/
├── package.json
└── README.md

## Installation

### Clone the repository
git clone https://github.com/vanshsingla66/Chatify.git
cd Chatify

### Install dependencies
npm run build

### Start the server
npm start

## Environment Variables

Create a `.env` file inside the backend folder:

MONGO_URI=your_mongodb_url
JWT_SECRET_KEY=your_secret
STREAM_API_KEY=your_key
STREAM_API_SECRET=your_secret
